# myFlix-client

## Description

Enable users to view information about movies. Allow the users to create a profile that lets them mark movies as favorites (or mark favorite movies as no longer a favorite).

## Build Information

To build and run the project, use the following command in the project root directory:

```bash
parcel src/index.html
```

## Project Dependencies

The project uses the following technologies:
* Bootstrap
* React
* React-Router
* Parcel
* Other supporting packages listed in the package.json file.

## API

The project uses the API documented [here](https://github.com/terryroe/movie-api/blob/main/public/documentation.html) and served from [here](https://users-movies-f50a18657028.herokuapp.com).
